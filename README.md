## 🎮 Game Rules

### Objective

The objective of the game is to correctly remember and repeat the sequence of blinking boxes. The sequence becomes longer with each level, making the game progressively more challenging.

### How the Game Works

1. The game starts when the player presses any key.
2. There are **four boxes** displayed on the screen.
3. One box will **blink** to start the sequence.
4. The player must **click the same box** that blinked.
5. In the next level, **another box will blink**, extending the sequence.
6. The player must repeat the **entire sequence from the beginning** in the correct order.
7. After each successful round, the sequence becomes **longer by one box**.

### Example

* Level 1:
  Box 1 blinks → Player clicks **Box 1**

* Level 2:
  Box 1 → Box 2 blink → Player clicks **Box 1 → Box 2**

* Level 3:
  Box 1 → Box 2 → Box 3 blink → Player clicks **Box 1 → Box 2 → Box 3**

### Game Over Condition

* If the player clicks the **wrong box**, the game ends.
* The final level reached becomes the **player's score**.

### Restart

* After the game ends, press any key to **restart the game**.
